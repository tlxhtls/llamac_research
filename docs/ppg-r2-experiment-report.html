LLaMAC PPG Emotion Regression R2 Report

PPG 단일 센서 rolling-window 피처로 연속형 감정 점수를 예측한 실험 요약입니다. 핵심 결론은 단순 window-random split에서는 R2가 높게 보이지만, trial 단위 누수를 제거한 정합 split에서는 ExtraTrees가 가장 좋았고 평균 test R2는 0.2083입니다.

작성일: 2026-05-12 Dataset: LLaMAC Figshare v6 Sensor: PPG only Runtime: Python 3.12 + uv

Experiment A: Window-random Sanity Check

먼저 window를 무작위 train/test로 나누는 optimistic sanity check를 수행했습니다. 이 방식은 같은 trial에서 나온 rolling window가 train과 test에 동시에 들어갈 수 있으므로 실제 일반화 성능으로 쓰면 안 됩니다.

Interpretation

  • PPG-only signal has measurable predictive information: coherent split R2 is above zero across all targets.
  • The effect size is modest: mean R2 0.2083 explains about 21% of held-out trial-level target variance.
  • The earlier R2 near 0.65 should not be used as a generalization claim because it comes from window-random splitting.
  • Among tested regressors, ExtraTrees is currently the best practical baseline for this PPG-only setup.
  • Participant-holdout evaluation is still expected to be harder and may reduce R2 further.
